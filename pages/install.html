Aviso!

A instalação do Flutter inclui o Dart, então caso for usar os dois, pule esta etapa.

Usando Chocolatey

1. Instale Chocolatey, um gerenciador de pacotes para Windows.

2. Abra o Powershell com permissões de Administrador.

3. Insira o comando para instalar:
choco upgrade dart-sdk

Para atualizar, use:
choco upgrade dart-sdk

Para desinstalar, use:
choco uninstall dart-sdk

a. Ubuntu e derivados (apt-get)

Instalação através do Terminal.

1. Faça uma atualização geral e instale o pacote de HTTP seguro.
sudo apt-get update && sudo apt-get install apt-transport-https

2. Baixe e instale a chave pública GPG da Google.
wget -qO- https://dl-ssl.google.com/linux/linux_signing_key.pub \ | sudo gpg --dearmor -o /usr/share/keyrings/dart.gpg

3. Adicione o repositório de pacote de Dart no seu sistema Linux.
echo 'deb [signed-by=/usr/share/keyrings/dart.gpg arch=amd64] https://storage.googleapis.com/download.dartlang.org/linux/debian stable main' \ | sudo tee /etc/apt/sources.list.d/dart_stable.list

4. Finalmente, instale Dart.
sudo apt-get update && sudo apt-get install dart

b. Debian e derivados (dpkg)

1. Baixe o pacote Debian do SDK de Dart

2. Abra o terminal no diretório do arquivo baixado.

3. Use o comando sudo dpkg para instalar o pacote *.deb.
sudo dpkg -i dart_3.4.0-1_amd64.deb
Se for necessário, substitue dart_3.4.0-1_amd64 pelo atual nome do pacote.

a. Usando Homebrew

1. Instale Homebrew, um gerenciador de pacotes para MacOS.

2. Adicione o tap oficial.
brew tap dart-lang/dart

3. Instale o Dart SDK com este comando.
brew install dart

Para atualizar, use:
brew upgrade dart

Para desinstalar, use:
brew uninstall dart

Usando Visual Studio Code

1. Instale no VSCode a extensão oficial de Flutter.

2. Pressione Ctrl + Shift + P para abrir a paleta de comandos.
a. Digite flutter na paleta de comandos.
b. Selecione Flutter: New Project.

3. VSCode o pedirá para localizar Flutter SDK.
a. Caso tiver o Flutter instalado, escolha Locate SDK para selecionar a pasta onde está instalado.
b. Se não o tiver instalado, escolha Download SDK.

4. Em "Which Flutter template?" ignore apertando Esc. Você poderá criar um projeto depois de instalado.

Baixando o Flutter

1. Em "Select Folder for Flutter SDK" está pedindo para especificar o local para armazenar o pacote.
O padrão é %USERPROFILE%. se quiser colocar em outro diretório, certifique-se de que o diretório não tenha espaço em seu nome, e não necessite de permissões de administrador.

2. Clique em Clone Flutter.

Espere o download.

3. Ao Terminar, clique em Add SDK to PATH.

4. Por ultimo, VSCode te informará da coleta de dados do Google.
Clique em Ok e reinicie o VSCode.

Através do Instalador

1. Baixe o instalador para Windows no site.

2. Abra o instalador e siga as instruções.

3. Cheque a instalação através do terminal:
a. Abra algum terminal.
b. Insira o comando go version.

Se o comando mostrar a versão de Go instalada, você está pronto para programar.

Extraindo pacote comprimido (tar.gz)

1. Baixe o pacote *.tar.gz para Linux no site.

2. Insira este comando no terminal:
rm -rf /usr/local/go && tar -C /usr/local -xzf go1.23.1.linux-amd64.tar.gz
O comando remove alguma possível instalação anterior de Go e extrai na mesma pasta a versão baixada.
Você talvez precise executar o comando como sudo ou trocar a versão especificada no comando.

3. Adicione Go ao PATH.
export PATH=$PATH:/usr/local/go/bin

3. Cheque a instalação através do terminal:
a. Abra algum terminal.
b. Insira o comando go version.

Se o comando mostrar a versão de Go instalada, você está pronto para programar.
