Introdução

Golang é uma linguagem recente que está com popularidade crescente. Go é uma linguagem muito simples para a programação de baixo nível, e isso contribui para que vários programadores adorem a linguagem.

Este é Gopher, o mascote da Golang.

Go é completamente separado de Flutter e Dart, sendo o seu próprio projeto. A Google criou a linguagem para que simplificasse e uniformizasse o código para facilitar a manutenção e o aprendizado de novos programadores. Por isso, Go é recomendado para programadores novos que querem embarcar na programação estática.
